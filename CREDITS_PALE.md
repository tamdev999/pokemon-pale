# Pokémon Pale — Credits & Acknowledgments

This project is an unofficial fan-made ROM hack. Pokémon is owned by Nintendo, Game Freak, and The Pokémon Company.

## Base Project

- **pokeemerald-expansion** by [RH Hideout](https://github.com/rh-hideout/pokeemerald-expansion)
  The decomp hack base this project is built on. Provides modern battle engine, Gen 1–9 species/moves/abilities, and extensive tooling.

- **pokeemerald** by [pret](https://github.com/pret/pokeemerald)
  The original Pokémon Emerald decompilation that makes source-level ROM hacking possible.

## Tools & Infrastructure

- **mGBA** by [Vicki Pfau](https://mgba.io/)
  Game Boy Advance emulator used for testing and Lua-scripted verification.

- **devkitARM** by [devkitPro](https://devkitpro.org/)
  GBA cross-compiler toolchain used to build the ROM.

## Inspiration & Methodology

- **Kyle Gill** — [Safari Gauntlet](https://github.com/gillkyle/pokemon-safari-gauntlet) / [Blog Post](https://kylegill.com/essays/building-pokemon-crystal-rom-hack-with-ai/)
  Pioneered the Lua verification harness approach for AI-assisted ROM hack development. Our testing infrastructure is directly inspired by his work.
