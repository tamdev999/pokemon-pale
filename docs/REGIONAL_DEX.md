# Pokémon Pale — Regional Pokédex

## Overview
The Pale Regional Dex is focused on Ghost, Psychic, Dark, and Fairy types, with thematic connections to death, spirits, the supernatural, and the boundary between life and death.

**Total families:** ~280 species across evolution lines
**Type focus:** Ghost, Psychic, Dark, Fairy (with supporting types via dual-typing)

## Notes
- Starters: Gastly, Litwick, Duskull (all Ghost-primary)
- Eevee line includes Umbreon (Dark), Sylveon (Fairy), Espeon (Psychic) only
- Nincada included specifically for Shedinja (Ghost)
- Several families span multiple type categories due to dual-typing or form changes
- Magikarp/Gyarados REMOVED from dex (no longer needed)
- Floette = ONLY Eternal Flower form (no Flabébé, no regular Floette, no Florges)
- Only the specific regional forms listed are included (e.g., only Alolan Marowak, not standard Marowak)
- Mega forms TBD — will be determined based on what's available for listed species
- All species NOT on this list are DISABLED in the build
